Convert integer to modulo-binary representation.

Given an integer x, returns an integer vector \(s = {2^{k_0} \times b, 2^{k_1} \times b, ..., 2^{k_T} \times b, r}\), where \(k_0 > ... > k_T \geq 0\), \(r < b\), and sum(s) = x.

Parameters
[in]xInteger of which the modulo-binary represention is computed.
[in]bInteger giving the binary modulus.
