Convert integer to modulo-binary representation.

Given an integer x, returns an integer vector \(s = {2^{k_0} \times base, 2^{k_1} \times base, ..., 2^{k_T} \times base, rem}\), where \(k_0 > ... > k_T \geq 0\), \(rem < base\), and sum(s) = x.

Parameters
[in]xInteger of which the modulo-binary represention is computed.
[in]baseInteger giving the binary modulus.
Returns
Integer vector containing the modulo-binary representation.
